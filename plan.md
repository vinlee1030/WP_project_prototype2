# Final Project – Midterm Plan

## Deploy Link
https://zombie-arena-v2.vercel.app

## 1. 本次 Prototype 已完成
- 完整的多人連線架構（P2P WebRTC 連線系統）
- 基本遊戲機制：玩家移動、射擊、生命值系統
- 殭屍 AI 系統：四種殭屍類型（Normal, Fast, Tank, Spitter）與自動生成
- 武器系統：七種武器（手槍、散彈槍、步槍、機槍、狙擊槍、火箭筒、火焰噴射器）
- 道具系統：補血包、彈藥、武器拾取、速度/傷害增益、護甲
- 波次系統：自動波次管理與難度遞增
- 即時聊天系統：遊戲內文字聊天
- 排行榜系統：即時顯示擊殺數與分數
- 跨平台支援：桌面版（鍵盤+滑鼠）與行動版（虛擬搖桿+觸控）
- 音效系統：Web Audio API 即時合成 8-bit 音效
- 視覺效果：粒子系統、爆炸效果、血跡特效
- 地圖系統：隨機生成地圖與障礙物
- 大廳系統：Google Sheets 整合顯示活躍房間

## 2. 最終版本預計完成項目
- **PvP 戰鬥模式**：團隊死鬥模式，支援多隊伍對戰（例如：玩家 vs 玩家 + 殭屍）
- **更智能的 AI**：殭屍具備更複雜的行為模式（包圍戰術、協同攻擊、路徑規劃）
- **美術與 UI 優化**：更精緻的像素藝術風格、動畫效果、UI/UX 改進
- **多樣化地圖**：多種主題地圖（城市、廢墟、實驗室等），每張地圖有獨特機制
- **多層建築系統**：可進入的建築物，多層樓設計，玩家可進行戰略性躲藏
- **角色系統**：多種角色選擇，每個角色有獨特技能與屬性
- **更多遊戲機制**：
  - 任務系統（每日任務、成就系統）
  - 武器升級系統
  - 特殊事件（空投補給、Boss 戰）
  - 天氣系統（影響視野與移動）
- **LLM 整合**：使用大型語言模型讓聊天室對話更真實，AI 助手提供遊戲提示
- **效能優化**：優化大量殭屍與玩家的渲染效能
- **遊戲平衡調整**：根據測試回饋調整武器傷害、殭屍強度、波次難度

## 3. 預期開發進度
- **Week 1**: 
  - 完成 PvP 戰鬥模式基礎架構（團隊分組、隊伍對戰邏輯）
  - 實作多層建築系統（建築物進入/離開、樓層切換、碰撞檢測）
  - 優化殭屍 AI（路徑尋找演算法、包圍戰術、協同行為）
  - 新增 2-3 張新地圖與主題設計

- **Week 2**: 
  - 完成角色系統（3-5 種角色，各具獨特技能）
  - 實作武器升級系統與特殊事件（空投、Boss 戰）
  - 整合 LLM 至聊天系統（使用 Gemini API 提供智能對話）
  - 美術資源優化（角色動畫、武器特效、UI 改版）
  - 效能優化（大量實體渲染優化、網路同步優化）

- **Week 3**: 
  - 完成任務與成就系統
  - 遊戲平衡測試與調整
  - 最終 UI/UX 打磨與動畫效果
  - 完整測試（多人連線壓力測試、跨平台測試）
  - 文件撰寫與部署優化
